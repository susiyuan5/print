# PrintLab 趋势雷达 Chrome 扩展

1. 确保本地工作台已启动：双击桌面 `print` 快捷方式。
2. 在 Chrome 地址栏打开 `chrome://extensions`，开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择本目录 `chrome-extension`。
4. 在当前 Chrome 打开任意榜单并手动完成登录或验证码。
5. 点击浏览器工具栏的“PrintLab 趋势雷达抓取”图标，选择“抓取具体项目并等待确认”。扩展只保留能识别为具体模型、商品或社媒内容的链接。
6. 返回本地趋势雷达页面，在“扩展待确认抓取”中打开预览并确认导入。

扩展只在你点击按钮时读取当前页面可见的具体项目链接、图片和说明；不会读取 Cookie、密码或登录令牌。非中文说明使用 Chrome 内置 Translator API 在本机翻译，首次使用某种语言时可能需要等待语言包下载。升级扩展后请在 `chrome://extensions` 点击扩展卡片上的“重新加载”。
